# محتويات الملف 
<br />
index.html(نقوم بعرض القيم المخزنه في قاعدة البيانات )
<br />
getValue.php(نقوم بسحب قيمة الحساس وحفظها في قاعدة البيانات )
<br />
showValue.php(هذا الملف خاص بعرض البيانات التي قمنا بتخزينها وعرضها )
