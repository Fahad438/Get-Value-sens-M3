# محتويات الملف 
<br />
1-index.html(نقوم بعرض القيم المخزنه في قاعدة البيانات )
<br />
2-getValue.php(نقوم بسحب قيمة الحساس وحفظها في قاعدة البيانات )
<br />
3-showValue.php(هذا الملف خاص بعرض البيانات التي قمنا بتخزينها وعرضها )
